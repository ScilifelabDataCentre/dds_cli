dds motd¶

[Super Admins only] Group command for managing Message of the Day within DDS.

dds motd [OPTIONS] COMMAND [ARGS]...

add¶

Add a new Message Of The Day.

[MESSAGE] is the MOTD that you wish do display to the DDS users.

dds motd add [OPTIONS] [MESSAGE]

Arguments

[MESSAGE]¶

Required argument

deactivate¶

[Super Admins only] Deactivate Message Of The Day.

dds motd deactivate [OPTIONS] [MOTD_ID]

Arguments

[MOTD_ID]¶

Required argument

ls¶

List all active MOTDs.

dds motd ls [OPTIONS]

send¶

Send motd as email to all users.

dds motd send [OPTIONS] [MOTD_ID]

Arguments

[MOTD_ID]¶

Required argument
